## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Install Node. [Node Version Manager](https://github.com/nvm-sh/nvm) is recommended.

```
nvm install <Latest Major Version Number>
nvm use <Latest Major Version Number>
```

Install `gatsby-cli`.

```
npm install -g gatsby-cli
```

### Installing

A step by step series of examples that tell you how to get a development env running

Clone the repository.

```
git clone https://github.com/Goff-Davis/portfolio-redesign
```

Start the gatsby development environment.

```
gatsby develop
```

## Linting

This project uses a combination of eslint and prettier to manage code styling and linting. This can be configured with the files `.eslintrc.js` and `.prettierrc`.

## Deployment

Gatsby can build for a production environment.

```
gatsby build
```

It is recommended to use Netlify as it will automatically pull from your git repository, build the site, and host it.
